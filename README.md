# 🌟 概述

MusicSquare 是一个简单的音乐搜索、下载和播放网站。
它提供了一个轻量级、浏览器友好的界面，可以直接从 GitHub Pages 站点搜索、播放和下载音乐。

GitHub Pages 部署地址：

👉 **在线访问:** <https://mumu99.github.io/musicMumu/>


## ✨ 主要功能

- 🎵 **在线音乐搜索与播放**  
  支持按关键词搜索歌曲，并直接在浏览器中播放。

- 📻 **多音乐源支持**  
  整合了咪咕、网易云、酷我和 QQ 音乐等在线平台。

- 💛 **可爱简洁的界面**  
  以皮卡丘为主题，界面清新可爱，适合轻松听歌。


## 🚀 使用方法

1. 打开在线网站：  
   👉 <https://mumu99.github.io/musicMumu/>

2. 在搜索框中输入歌手名或歌曲名（例如："周杰伦"）。

3. 点击搜索结果即可开始播放，享受音乐。


## 💻 开发与自定义

- 本站点通过 **GitHub Pages** 托管。
- 你可以自定义：
  - 主题（颜色、背景、图标）
  - API 接口（例如：自建咪咕、网易云、酷我和 QQ 音乐的代理/桥接服务器）
  - 播放器行为（自动播放、播放列表、歌词面板等）

更多详情请查看本仓库中的源代码文件，根据需要调整 HTML / CSS / JavaScript。
